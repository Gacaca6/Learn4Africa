# Learn4Africa — Remotion

A 3-minute, 1920×1080 product demo video, rendered via [Remotion](https://www.remotion.dev).

Ported verbatim from the `Learn4Africa Demo.html` HTML/React prototype. All 10 scenes, easings, animations, copy, and timing are unchanged.

## Install

```bash
cd remotion
npm install
```

## Preview (Remotion Studio)

```bash
npm run dev
```

Opens a browser with a scrubbable timeline.

## Render to MP4

```bash
npm run build
```

Output: `remotion/out/learn4africa.mp4`
